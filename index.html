Virat Kohli ODI Performance Dashboard

This Power BI dashboard provides a comprehensive analysis of Virat Kohli's performance in One Day Internationals (ODIs). It highlights key statistics such as total runs, batting average, strike rate, centuries, and consistency across years. The dashboard allows users to explore his performance against different teams. Built with interactive visuals and filters, the dashboard enables dynamic exploration of trends and insights. It reflects the practical application of data analytics in sports, showcasing how performance metrics can be translated into meaningful stories through visualization.
